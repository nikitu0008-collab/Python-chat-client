# 💬 Python TCP Chat Client

Простой консольный клиент для TCP-чата на Python. Использует многопоточность: один поток слушает сервер, другой - отправляет сообщения.

## 🚀 Возможности

- Подключение к серверу по IP и порту
- Обмен текстовыми сообщениями в реальном времени
- Команда `/exit` для выхода
- Корректная обработка разрыва соединения

## 🛠️ Требования

- Python 3.7+
- Никаких внешних библиотек (только стандартный `socket` и `threading`)

## 📦 Установка и запуск

```bash
git clone https://github.com/nikitu0008-collab/python-chat-client.git
cd python-chat-client
python client.py
🧪 Пример работы
text

🧠 Как это устроено
Создаётся TCP-сокет и подключается к серверу.

Запускается фоновый поток для приёма сообщений, чтобы не блокировать ввод.

Основной поток читает ввод пользователя и отправляет данные на сервер.

Сервер должен быть запущен заранее (не входит в этот репозиторий).

📝 Примечания
Код написан в учебных целях, для демонстрации работы с сокетами и потоками в Python.

Серверная часть не включена (может быть отдельным репозиторием).
